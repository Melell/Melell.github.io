About me

Hello! I am a passionate and driven C++ game programmer, who is interested in working on either gameplay, tools or engine programming. This webpage showcases some of the games and personal projects I have worked on so far. Hope you enjoy! :)

Shadow Racer

Shadow Racer is a third-person driving adventure game set in a post-apocalyptic Grand Canyon. The player must find and use the ancient powers of the flash, turbo and bumper in order to defeat the cars from the darkness empire. It was published on Steam in May 2022.

Learn more

Seavilizations

Seavilizations is a 3D multiplayer boat-combat game in which all the players have to fight against each other to obtain a certain number of chalices so that they can escape from the map to win.

Learn more

Entails

Entails is a 2D platformer, shooter game where the player must disinfect a human body. The player must go through parts of the human body, fighting both the different viruses and the body's defenses.

Learn more

Projects

These are some other projects that I have developed either at DigiPen, or in my free time. They cover topics, like animation, AI, graphics, optimizations, physics and networking.

Animation Engine + Cloth Simulation

In this project, I implemented a small animation engine from scratch using C++. It is capable of spline creation and interpolation, 1D and 2D animation blending, IKs, and simulation of cloth.

Learn more

Ant Colony Simulation

Two person AI research project. Implemented an ant colony foraging behaviour simulation of up to 1000 ants, inspired by the Ant Colony Optimization algorithm, from scratch using C++.

Learn more

Space Partitioning

In this C++ project, I did several performance improvements to the collision detection system through spacial acceleration structures like octrees or kd-trees. I also implemented the GJK algorithm for collision detection between arbitrary convex shapes.

Learn more

Networked Asteroids Game

Simple networked asteroids game made from scratch with C++ and the Winsock library. It uses a server-client model, supports up to 4 players, and uses UDP to send packets.
